Plataforma de Apoio ao Turismo Local

Plataforma web desenvolvida em ASP.NET MVC (C#) para apoiar a gestão de alojamentos locais, permitindo divulgar anúncios, gerir reservas e oferecer ao cliente uma experiência simples e consistente de pesquisa e compra.

Funcionalidades Principais

- Pesquisa de alojamentos com filtros e seleção de datas
- Checkout unificado e gestão completa do estado das reservas
- Área do cliente com histórico, favoritos e avaliações
- Dashboard do administrador com KPIs e estatísticas
- Gestão de anúncios (preços, imagens, descontos, localização no mapa)
- Mensagens internas entre cliente e gestor
- Calendário de disponibilidades
- Notificações internas
- Internacionalização (PT/EN/ES/FR)

- MVC (Model–View–Controller)
- EF Migrations para evolução da base de dados
- Organização modular por áreas: Anúncios, Reservas, Utilizadores, Dashboard, Mensagens

Como Executar

- Clonar o repositório
- Abrir a solução no Visual Studio
- Configurar a connection string para o SQL Server
- Executar Update-Database (EF Migrations)
- Iniciar com IIS Express
